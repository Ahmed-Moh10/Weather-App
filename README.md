# 🌤️ Weather App

A responsive weather web application that allows users to search for any city and view current weather conditions along with a 3-day forecast.

---

## 🚀 Features

* 🔍 Search for any city worldwide
* 📍 Detect user's current location automatically
* 🌡️ Display current temperature and weather condition
* 📅 3-day weather forecast
* 💨 Wind speed, humidity, and direction
* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* 🎨 Clean UI using Bootstrap

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (Vanilla JS)
* Weather API

## ⚙️ How It Works

1. User enters a city name in the search input
2. App fetches data from **Weather API**
3. Displays:

   * Current weather
   * Next 2 days forecast
4. Automatically detects user's location on first load
